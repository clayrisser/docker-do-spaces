## Installation

```sh
docker run -p 8080:8080 \
  -e BUCKET=some-bucket \
  -e ENDPOINT=sfo2.digitaloceanspaces.com \
  -e PREFIX=some/prefix \
  codejamninja/do-spaces:latest
```
